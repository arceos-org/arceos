Error code definition used by ArceOS.

It provides two error types and the corresponding result types:

  • AxError and AxResult: A generic error type similar to std::io::ErrorKind.
  • LinuxError and LinuxResult: Linux specific error codes defined in errno.h. It can be converted from AxError.
